#  Event Management System

##  Overview

This is a full-stack **Event Management System** where users can create, manage, and explore events. The application includes authentication, responsive UI, and smooth user experience.

---

##  Features

###  Event Management (CRUD)

* Create Events
* View Events
* Update Events
* Delete Events

###  Authentication System

* User Signup
* User Login
* Secure API integration

###  Responsive UI
* Fully mobile-friendly 
* Optimized for desktop 
* Smooth animations & modern design

###  Additional Features

* Testimonials section (auto slider)
* Gallery with auto-changing images
* Contact form with API integration

---

##  Tech Stack

### Frontend:

* React.js
* Tailwind CSS

### Backend:

* Node.js
* Express.js

### Database:

* MongoDB
